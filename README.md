# DQN

##### Usage
```python <FILE> --[options] ```

##### Files
- [```dqn_cartpole.py```](dqn_cartpole.py) DQN agent for CartPole environment
- [```dqn_breakout.py```](dqn_breakout.py) DQN agent for Breakout environment
- [```dqn_doom.py```](dqn_doom.py) DQN agent for Doom environment

##### Limitations
- Breakout provides suboptimal solution
- Doom currently not compatible with keras-rl
